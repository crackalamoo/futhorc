How to Write in Runes

First of all, forget everything you know about English spelling. My system of Runic spelling is (mostly) phonetic, unlike J. R. R. Tolkien's system, which mostly substitutes the non-phonetic English spelling system letter by letter. There are a few exceptions to this phonetic system, which I will explain later. First, here is a basic guide to writing modern English in my system of runes. I have included the sounds of American English as I speak it in IPA (the international phonetic alphabet), plus an example of this sound in a word, and finally the rune to represent that sound. I start with pure vowels, then diphthongs, and finally consonants. There is no distinction for vowel length.

You may have noticed that some runes stand for more than one sound. There are a few other possible ambiguities. Here are some rules to clear that up.

  1. Apostrophes and other punctuation marks are used just like in standard English.
  2. If there is ambiguity between /i/ and /ɪ/, use ᛁᛁ for /i/ and ᛁ for /ɪ/. So you have we'll/ᚹᛁ'ᛚ, will/ᚹᛁᛚ, wheel/ᚹᛁᛁᛚ. This applies even when "wheel" rather than "will" can be inferred from the context, since "will" is a word. The same context-independent idea applies for the following rules. Also note that if there is a proper noun that can be written with the "simpler" spelling (as in the table above) you don't have to worry about it. So "Sid" and "seed" are both spelled ᛋᛁᛞ, even though you would expect ᛋᛁᛁᛞ for "seed" if "Sid" were not a proper noun. Also note that ᛁᛁ may represent /iɪ/ as in being/ᛒᛁᛁᛝ.
  3. If there is ambiguity between /f/ and /v/, use ᚠ for /f/ and ᚡ for /v/. So you have live/ᛚᛁᚠ, leaf/ᛚᛁᛁᚠ, leave/ᛚᛁᛁᚡ, lives/ᛚᛁᚠᛋ, leaves/ᛚᛁᛁᚠᛋ. Note that rules apply in the order they are listed here in case of a conflict, as you can see here between "live" and "leaf".
  4. If there is ambiguity between /s/ and /z/, use ᛋᛋ for /s/ and ᛋ for /z/. So you have ones/ᚹᚢᚾᛋ, once/ᚹᚢᚾᛋᛋ.
  5. If there is ambiguity between /θ/ and /ð/, use ᚦ for /θ/ and ᚧ for /ð/. I can't think of any examples for this, this should be rare.
  6. "No" is spelled ᚾᚩ and "know" is spelled ᚾᚩᚹ.
  7. Words which use "tr" for /tʃɹ/ in standard English are spelled with ᛏᚱ, not ᚳᚱ. Similar for "dr" and /dʒɹ/. So you have truck/ᛏᚱᚢᛣ, draw/ᛞᚱᛟ.
  8. In fast speech, vowels are sometimes converted to schwas. This shouldn't be reflected in writing. So you have comma/ᛣᛟᛗᚪ (not ᛣᛟᛗᛇ), and remember/ᚱᛁᛗᛖᛗᛒᛇᚱ (not ᚱᛇᛗᛖᛗᛒᛇᚱ).
  9. Syllabic consonants are spelled with ᛇ before the consonant. So you have bottle/ᛒᛟᛏᛇᛚ.
  10. ᛝ strictly represents /ŋ/, not /ŋg/. So you have angle/ᚫᛝᚸᛇᛚ, not ᚫᛝᛇᛚ.
  11. ᛋ and ᛏ are written together as the ligature ᛥ as in stone/ᛥᚩᚾ.
  12. The name of this alphabet is written ᚠᚢᚦᚩᚱᚳ, but pronounced /fuθork/ like "FOO-thork" as if it were spelled ᚠᚣᚦᚩᚱᛣ.
  13. Ash is spelled ᚫᛋᚳ, not ᚫᛋᚻ. Fish is spelled ᚠᛁᛋᚳ, not ᚠᛁᛋᚻ. This extends to derived words too, so you have ashy/ᚫᛋᚳᛁ and fishing/ᚠᛁᛋᚳᛁᛝ.
